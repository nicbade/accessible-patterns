Notifications

Keyboard-only navigation

  • Use Tab/Shift+Tab to move between fields.
  • Use Space to check/uncheck boxes; Arrow keys to change radio choices.
  • Use Enter to submit when focused on the Submit button.

JAWS / NVDA navigation

  • Use form mode (auto when entering a form). If needed: NVDA toggle with NVDA+Space; JAWS toggle Virtual PC Cursor with Insert+Z.
  • JAWS/NVDA announce labels from <label for=…>; additional guidance comes from aria-describedby (e.g., “All fields required”).
  • Errors should be announced via role="alert" or associating messages with the field (you can add later).
